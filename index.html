About This Project

Project Inspiration

The project inspiration for this image compression tool came from a personal experience of struggling every time I wanted to resize and share high-resolution images online or for an application. The constant frustration of longer load times, large file sizes after resizing, and not maintaining picture quality, inspired my teammate and I for the creation of a tool that could compress images efficiently without compromising image quality. The goal was to make this tool accessible for everyone, from photographers, students, researchers and to web developers, ensuring smoother workflows and better user experience.

This project is also part of my portfolio for the Holberton School, where we are encouraged to develop practical, impactful tools. Over a span of a few weeks, I worked closely with my teammate to make this idea a reality, learning and implementing advanced image processing techniques along the way.
